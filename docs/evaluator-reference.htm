Evaluator Reference

export function addElementUpdateBinding( name, element )
Binds an HTML element property or attribute to a variable.
name : Name of the bound variable.
element : Reference to the bound HTML element.
export function addFunctionBindings ( name, functions )
Binds a variable to a list of functions. The set() function executes them if the value of the bound variable is changed.
name : Name of the bound variable.
functions : Array of function names or a string with a comma-separated list of function names.
export function addPropertyUpdateBinding ( name, objectReference, propertyName )
Binds a variable to an object property. The set() function uses this association to update bound properties with a new value.
name : Name of the bound variable.
objectReference : The target object to be updated.
propertyName : The name of the property to update.
data-attrib=attributeName
Binds an element attribute to a variable.
attributeName : Name of the attribute.
data-get=variableName
Attribute of an HTML element, indicates a read-only binding.
variableName : Name of the bound variable.
data-member=propertyName
Binds an element property to a variable.
propertyName : Name of the bound property.
data-set=variableName
Attribute of an HTML input element, indicates a read-write binding.
variableName : Name of the bound variable.
export function initPage ( )
Creates the binding information and adds change event handlers to input elements.
Must be called from the HTML document.
export function inputChangeHandler( evt )
Called when the change event of a bound HTML input elements is fired. Note that programmatic value change from a script does not fire the event. Calls the set() function to update the value of the bound variable.
evt : Change event parameter object
export function set ( name, value )
Sets the value of a bound global variable, updates bound properties and calls bound functions.
name : Name of the bound variable
value : New value for the variable
